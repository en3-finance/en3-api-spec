# Public/Private Boundary

Status: public reference / sandbox artifact. This repository is intended to document and demonstrate the En3 integration surface. Production cryptography, signing orchestration, policy enforcement, risk logic, ledger infrastructure, treasury execution, and customer deployments are private by design.

## Public

- Sandbox resource names, request shapes, response shapes, and status values.
- Canonical public webhook event names.
- Synthetic SandBank examples and deterministic sandbox identifiers.
- Public error envelope and sandbox error codes.
- Wallet and transaction lifecycle sequencing.

## Private

- Production cryptography and custody implementation.
- Production signing orchestration.
- Production policy enforcement and risk logic.
- Ledger infrastructure and treasury execution.
- Real network endpoints, keys, secrets, deployment configuration, customer context, or private business context.

## SandBank

SandBank is fictional and synthetic. It exists only to demonstrate the sandbox integration surface.
